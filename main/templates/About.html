Why was this website made?

To develop a working knowledge of full-stack web development. Going into the project I could use python and VBA, that's it. No bootcamp. In the search of project ideas, I came across PlaylistMiner. It inspired me to create a clone with the same functionality because it's a simple project. I could focus on the learning the various aspects over the engineering complexity. Also, to make awesome playlists really quickly!

Did I learn?

Yup, through trial and error.
HTML, CSS, Flask, RESTful API, SQL, GIT,MVC, OAuth2.0, HTTP requests, Cloud hosting + deployment
Focused on writing clean, modular, testable code.
For anyone curious in the general steps, I wrote some details about my process in this quick article.
